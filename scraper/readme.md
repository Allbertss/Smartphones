# Breezy smartphones scraper

A Python-based web scraper to extract 2nd life smartphones from https://breezy.pl

Config in [src/BreezyScraperConfig.py](src/BreezyScraperConfig.py)

```bash
cd scraper
```

To run the scraper
```bash
python3 src/main.py
```

To run the tests
```bash
python3 -m pytest
```
